# TypeScript-Babel-Express-Starter

### Requirements

* Node.js 12 LTS

### Setup

```shell script
$ nvm use
$ npm i
```

### Static Type Checking

```shell script
$ npm run type-check
```

### Running Tests
```shell script
$ rm -r dist
$ npm t
```

### Building JavaScript code
```shell script
$ npm run build:js
```

Built code will be put in dist folder.

### Starting App

```shell script
$ npm start
```
