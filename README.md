# TypeScript-Babel-Express-Starter

### Requirements

- Node.js 14

### Setup

```shell
npm i
```

### Static Type Check

```shell
npm run type-check
```

### Run Tests

```shell
npm t
```

### Build JavaScript code (for production)

```shell
npm run build:js
```

Built code will be put in `dist` folder. Execute it with:

```shell
node dist
```

### Start App (for development)

```shell
npm start
```
