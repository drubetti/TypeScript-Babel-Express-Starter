# TypeScript-Babel-Express-Starter

### Requirements

- Node.js

### Setup

```shell
npm i
```

### Static Type Check

```shell
npm run type-check
```

### Run Tests

```shell
npm t
```

### Build JavaScript code

```shell
npm run build:js
```

Built code will be put in `dist` folder.

### Start App (AOT compilation)

Build JavaScript code, then run:

```shell
node dist
```

### Start App (on-the-fly compilation)

```shell
npm start
```
