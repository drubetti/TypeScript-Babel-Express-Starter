# TypeScript-Babel-Express-Starter

### Requirements

- Node.js 14 LTS

### Setup

```shell script
$ npm i
```

### Static Type Checking

```shell script
$ npm run type-check
```

### Running Tests

```shell script
$ npm t
```

### Building JavaScript code

```shell script
$ npm run build:js
```

Built code will be put in `dist` folder.

### Starting App

```shell script
$ npm start
```
